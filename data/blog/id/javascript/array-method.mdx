Array merupakan tipe data yang sangat penting dan pasti ada pada setiap bahasa pemrograman. Javascript memiliki metode yang memungkinkan kita untuk memanipulasi dan mengolah array menjadi mudah. Terdapat banyak method yang ada dalam javascript yang sering digunakan untuk memanipulasi array seperti `map`, `filter`, `reduce`, `find`, `findIndex`, `push`, `pop`, `shift`, `unshift`, `slice`, `splice`, `forEach`, `some` dan `every`. Mari kita bahas satu persatu.

## Metode Map

Jika kita ingin mengubah setiap elemen yang kita miliki dalam array, kita bisa menggunakan `map()`. Metode ini akan menghasilkan array baru yang elemen nya sudah berubah sesuai dengan apa yang kita mau. Sebagai contoh kita memiliki array yang berisi beberapa angka dan kita ingin mengkalikan dengan 2 pada setiap elemen yang kita miliki didalam array. Metode ini akan sangat berguna jika kita ingin mengubah semua elemen yang ada dalam array kita tanpa mengubah ukuran dari array yang kita miliki.

```javascript
const numbers = [1, 2, 3, 4]
const doubled = numbers.map((num) => num * 2)
// Hasil: [2, 4, 6, 8]
```

Selain itu, metode `map()` ini akan sangat berguna jika kita ingin mengekstrak data tertentu. sebagai contoh kita memiliki objek `id` dan `nama` yang ada dalam array.

```javascript
const users = [
  { id: 1, name: 'John' },
  { id: 2, name: 'Jane' },
  { id: 3, name: 'Doe' },
]
const userIds = users.map((user) => user.id)
// Hasil: userIds: [1, 2, 3]
```

## Metode Filter

Metode `filter()` digunakan jika kita ingin menghasilkan array baru yang berisi elemen-elemen yang memenuhi kriteria tertentu. Setiap elemen akan diproses melalui fungsi callback yang akan mengembalikan nilai true atau false, yang akan menentukan apakah elemen tersebut layak dimasukkan ke dalam array baru.

```javascript
const ages = [15, 25, 35, 12, 40]
const adults = ages.filter((age) => age >= 18)
// Hasil: [25, 35, 40]
```

Pada contoh diatas, kita ingin mencetak sebuah array umur yang yang baru dan memiliki element umur sama atau diatas 18. Maka kita menerapkan kondisi `age >= 18`.

## Metode Reduce

Metode `reduce()` memungkinkan kita menggabungkan semua elemen array menjadi satu nilai tunggal. Fleksibilitas metode ini membuatnya berguna untuk berbagai keperluan, mulai dari menjumlahkan angka hingga membuat struktur data kompleks. Nilai akumulator akan diproses bersama setiap elemen array untuk menghasilkan output akhir.

```javascript
const prices = [10, 20, 30]

const total = prices.reduce((sum, price) => sum + price, 0)

// Hasil: 60
```

Pada contoh di atas, nilai awal akumulator adalah 0, kemudian setiap elemen dalam array ditambahkan ke akumulator hingga menghasilkan total 60.

## Metode Find dan FindIndex

Berikut metode yang membantu menemukan elemen tertentu dalam array:

- `find()` mengembalikan elemen pertama yang cocok dengan kondisi, sangat berguna untuk mencari item unik.

- `findIndex()` mengembalikan posisi (indeks) dari elemen pertama yang cocok, berguna saat kita perlu tahu lokasi elemen dalam array.

```javascript
const users = [
  { id: 1, name: 'John' },

  { id: 2, name: 'Jane' },
]

const john = users.find((user) => user.name === 'John')

const johnIndex = users.findIndex((user) => user.name === 'John')
```

## Push, Pop, Shift, dan Unshift

Metode-metode ini memodifikasi array asli dengan menambah atau menghapus elemen dari salah satu ujungnya. Sangat penting untuk operasi stack dan queue dalam JavaScript:

```javascript
const fruits = ['apple']

fruits.push('banana') // Menambahkan ke akhir: ['apple', 'banana']

fruits.pop() // Menghapus dari akhir: ['apple']

fruits.unshift('orange') // Menambahkan ke awal: ['orange', 'apple']

fruits.shift() // Menghapus dari awal: ['apple']
```

## Slice dan Splice

Dua metode yang kuat untuk memanipulasi bagian array dengan tujuan berbeda:

```javascript
// Slice: Mengekstrak bagian array tanpa memodifikasinya

const numbers = [1, 2, 3, 4, 5]

const subset = numbers.slice(1, 3) // [2, 3]

// Splice: Mengubah array dengan menghapus/mengganti elemen

const letters = ['a', 'b', 'c']

letters.splice(1, 1, 'x') // Menggantikan 'b' dengan 'x'
```

## Metode forEach

Metode ini memungkinkan kita menjalankan kode untuk setiap elemen array tanpa membuat array baru. Ideal untuk operasi yang tidak memerlukan nilai kembalian, seperti mencatat informasi atau memperbarui variabel eksternal.

```javascript
const items = ['book', 'pen', 'desk']

items.forEach((item, index) => {
  console.log(`Item ${index + 1}: ${item}`)
})
```

## Metode Some dan Every

Metode untuk menguji elemen array menggunakan kondisi boolean:

- `some()`: Memeriksa apakah minimal satu elemen memenuhi kondisi yang diberikan

- `every()`: Memeriksa apakah semua elemen memenuhi kondisi yang diberikan

```javascript
const scores = [75, 80, 95]

const hasPassingScore = scores.some((score) => score >= 90) // true

const allPassed = scores.every((score) => score >= 70) // true
```
