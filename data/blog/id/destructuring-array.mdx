Jika sebelumnya kita melakukan destrukturisasi pada objek, kita juga bisa melakukan destrukturisasi pada array.

## Dasar Destrukturisasi Array

Ekstrak nilai dari array ke dalam variabel terpisah:

```js
const fruits = ['apple', 'banana', 'grape', 'orange']

// Destrukturisasi
const [first, second, third, fourth] = fruits

console.log(first) // "apple"
console.log(second) // "banana"
```

## Teknik Destrukturisasi Lanjutan

### Melewati Elemen

Lewati elemen array yang tidak diinginkan menggunakan koma:

```js
const fruits = ['apple', 'banana', 'grape', 'orange']
const [first, , third] = fruits

console.log(first) // "apple"
console.log(third) // "grape"
```

### Operator Rest

Kumpulkan elemen yang tersisa menggunakan operator rest (...):

```js
const fruits = ['apple', 'banana', 'grape', 'orange']
const [first, second, ...remaining] = fruits

console.log(first) // "apple"
console.log(remaining) // ["grape", "orange"]
```

### Nilai Default

Berikan nilai default untuk elemen yang tidak ada atau undefined:

```js
const colors = ['red']
const [primary, secondary = 'blue'] = colors

console.log(primary) // "red"
console.log(secondary) // "blue" (nilai default digunakan)
```

### Destrukturisasi Array Bersarang

Ekstrak nilai dari array yang bersarang (nested arrays):

```js
const fruits = ['apple', ['red', 'green'], 'banana']
const [name, [color1, color2]] = fruits

console.log(name) // "apple"
console.log(color1) // "red"
console.log(color2) // "green"
```

### Kombinasi dengan Destrukturisasi Objek

Destrukturisasi bisa dikombinasikan antara array dan objek:

```js
const data = ['apple', { color: 'red', weight: 150 }]
const [name, { color, weight }] = data

console.log(name) // "apple"
console.log(color) // "red"
console.log(weight) // 150
```

### Destrukturisasi Parameter Fungsi

Destrukturisasi array langsung pada parameter fungsi:

```js
function analyzeFruits([first, second, ...others]) {
  console.log(`Buah pertama: ${first}`)
  console.log(`Buah kedua: ${second}`)
  console.log(`Buah lainnya: ${others.join(', ')}`)
}

analyzeFruits(['apple', 'banana', 'grape', 'orange'])
```

### Menukar Nilai

Tukar nilai antar variabel dengan mudah:

```js
let a = 'apple'
let b = ('banana'[
  // Tukar nilai
  (a, b)
] = [b, a])

console.log(a) // "banana"
console.log(b) // "apple"
```

### Penanganan Error

Destrukturisasi dengan array kosong atau undefined harus diperhatikan:

```js
// Contoh 1: Dengan array kosong dan nilai default
const [first = 'default'] = []
console.log(first) // "default"

// Contoh 2: Error handling saat destrukturisasi null
try {
  const [a, b] = null // Akan menghasilkan TypeError
} catch (error) {
  console.error('Hati-hati saat melakukan destrukturisasi null/undefined')
}
```

### Destrukturisasi dengan Iterables Lainnya

Destrukturisasi bisa digunakan untuk iterable lain, bukan hanya array:

```js
// String iterable
const [a, b, c] = 'xyz'
console.log(a, b, c) // "x" "y" "z"

// Set iterable
const [first, ...rest] = new Set([1, 2, 3])
console.log(first, rest) // 1, [2, 3]
```

## Contoh Penggunaan Praktis

### 1. Pengembalian Fungsi

```js
function getCoordinates() {
  // Implementasi yang jelas
  return [10, 20, 30] // x, y, z coordinates
}

const [x, y, z] = getCoordinates()
console.log(x, y, z) // 10, 20, 30
```

### 2. Pemrosesan Array

```js
const numbers = [1, 2, 3, 4, 5]
const [head, ...tail] = numbers

console.log(head) // 1
console.log(tail) // [2, 3, 4, 5]
```

### 3. Penugasan Beberapa Nilai

```js
const dateString = '2024-09-29'
const [year, month, day] = dateString.split('-')

console.log(`Tanggal: ${day}, Bulan: ${month}, Tahun: ${year}`)
```

### 4. React Hooks

```js
// Penggunaan destrukturisasi array dalam React Hooks
const [count, setCount] = useState(0)
const [isLoading, setIsLoading] = useState(false)

// Custom hooks juga menggunakan pola yang sama
const [data, loading, error] = useCustomData()
```

### 5. Pattern Matching

```js
// Menggunakan destrukturisasi untuk pattern matching
function processAction(actionData) {
  if (Array.isArray(actionData) && actionData.length >= 2) {
    const [action, payload] = actionData

    if (action === 'ADD_USER') {
      // Proses penambahan user dengan payload
      return `User ${payload.name} ditambahkan`
    }
  }
  return 'Format aksi tidak valid'
}

console.log(processAction(['ADD_USER', { name: 'John' }]))
```

## Pertimbangan Performa

Meskipun destrukturisasi sangat berguna, perlu diperhatikan beberapa hal terkait performa:

- Hindari destrukturisasi berlebihan dalam loop dengan jumlah iterasi besar
- Untuk operasi berulang pada data yang sama, simpan hasil destrukturisasi di luar loop
- Jika hanya memerlukan satu atau dua item dari array besar, pertimbangkan akses indeks langsung

```js
// Kurang efisien dalam loop besar
for (let i = 0; i < 10000; i++) {
  const [a, b] = getBigArray() // Overhead jika dilakukan berulang
}

// Lebih efisien
const bigArray = getBigArray()
const [a, b] = bigArray // Destrukturisasi sekali
for (let i = 0; i < 10000; i++) {
  // Gunakan a dan b tanpa destrukturisasi lagi
}
```

## Praktik Terbaik

✅ Lakukan:

- Gunakan destrukturisasi untuk mengembalikan beberapa nilai
- Beri nama variabel secara jelas berdasarkan isinya
- Gunakan operator rest untuk fleksibilitas
- Destrukturisasi parameter fungsi ketika berurusan dengan array
- Selalu berikan nilai default untuk elemen yang mungkin undefined

❌ Hindari:

- Destrukturisasi array yang sangat bersarang (lebih dari 2-3 level)
- Menggunakan terlalu banyak slot kosong (elemen yang dilewati)
- Terlalu banyak destrukturisasi ketika hanya beberapa nilai yang dibutuhkan
- Melakukan destrukturisasi berulang kali dalam loop besar
