Destrukturisasi objek merupakan fitur yang ampuh untuk mengekstrak properti dari objek secara efisien. Alih-alih mengakses nya satu persatu, dengan destrukturisasi membuat syntax kita lebih rapih dan mudah dibaca.

Sesuai dengan istilah nya, destrukturisasi pada objek merupakan proses memecah properti dari objek dan memasukkan nya kedalam variabel yang terpisah.

Anggap lah kita memiliki variabel apple yang memiliki object `name`, `color`, `weight` dan `taste` seperti berikut:

```js
const apple = {
  name: 'Apple',
  color: 'Red',
  weight: 150,
  taste: 'Sweet',
}
```

Tanpa destrukturisasi, kita perlu mendeklarasikan satu persatu dari properti apple untuk mengakses nilai setiap properti.

```js
let name = apple.name // output: Apple

let color = apple.color // output: Red

let weight = apple.weight // output: 150

let taste = apple.taste // output: Sweet
```

Tentu ini akan sangat menguras tenaga jika kita berurusan dengan objek yang sangat banyak atau bahkan nested object seperti jika kita sedang berurusan dengan respon JSON pada API.

## Dasar dari Object Destructuring

Lalu bagaimana kita melakukan destrukturisasi pada objek apple yang telah kita buat tadi? kita bisa melakukan nya hanya dengan satu baris kode saja.

```js
const { name, color, weight } = apple

console.log(name) // "Apple"
console.log(color) // "Red"
```

Dengan begitu, `name`, `color` dan `weight` memiliki nilai yang sama objek dari apple.

## Advanced Destructuring Techniques

Destrukturisasi juga bisa kita gunakan dalam case-case tertentu seperti:

### 1. Renaming Variables

Kita bisa mengekstrak properti dengan nama variabel yang kita inginkan:

```js
const { name: fruitName, color: fruitColor } = apple

console.log(fruitName) // "Apple"
console.log(fruitColor) // "Red"
```

### 2. Default Values

Anggaplah kita ingin mengambil properti size dalam variable apple:

```js
console.log(size) // undefined
```

Karena didalam variabel apple tidak ada properti size, maka value akan muncul sebagain `undefined`. Kita bisa menambahkan default value kedalam variabel size.

```js
const { size = 'Medium' } = apple

console.log(size) // "Medium"
```

### 3. Nested Object Destructuring

Kita juga bisa mengekstrak nilai dari nested object. ini akan sangat berguna jika kita berhubungan dengan respon JSON sebuah API.

```js
const fruit = {
  name: 'Apple',
  nutrients: {
    vitamin_c: 4.6,
    fiber: 2.4,
  },
}

const {
  nutrients: { vitamin_c, fiber },
} = fruit

console.log(vitamin_c) // 4.6
console.log(fiber) // 2.4
```

### 4. Function Parameter Destructuring

Kita bisa menggunakan destrukturisasi dalam fungsi sebagai parameter.

```js
function describeFruit({ name, color, taste }) {
  return `The ${name} is ${color} and tastes ${taste}.`
}

describeFruit(apple) // "The Apple is Red and tastes Sweet."
```

## Conclusion

Destrukturisasi Objek mempermudah kita untuk mengakses properti dan membuat kode javascript kita lebih maintainable. Dan ini akan sangat berguna ketika kita mengolah data JSON dalam API atau struktur data yang kompleks.
