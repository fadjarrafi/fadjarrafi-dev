JavaScript arrays come with powerful built-in methods that make data manipulation easier. These methods are fundamental tools for modern JavaScript development, helping you write more concise and maintainable code. This guide covers the most important array methods you'll use in your projects.

## Map Method

The `map()` method creates a new array by transforming each element. It's perfect for when you need to perform the same operation on every element in an array while keeping the array length the same. The original array remains unchanged, following JavaScript's immutability principles.

```javascript
const numbers = [1, 2, 3, 4]
const doubled = numbers.map((num) => num * 2)
// Result: [2, 4, 6, 8]
```

## Filter Method

Use `filter()` to create a new array with elements that pass a test. This method is invaluable when you need to extract specific elements based on certain conditions. Each element is passed through a callback function that returns true or false, determining whether the element should be included in the new array.

```javascript
const ages = [15, 25, 35, 12, 40]
const adults = ages.filter((age) => age >= 18)
// Result: [25, 35, 40]
```

## Reduce Method

`reduce()` combines array elements into a single value. This versatile method can be used for everything from summing numbers to creating complex data structures. The method takes an accumulator value and processes each array element to build the final result.

```javascript
const prices = [10, 20, 30]
const total = prices.reduce((sum, price) => sum + price, 0)
// Result: 60
```

## Find and FindIndex Methods

These methods help you locate specific elements in an array:

- `find()` returns the first matching element, perfect for finding unique items
- `findIndex()` returns the first matching element's index, useful when you need the position

```javascript
const users = [
  { id: 1, name: 'John' },
  { id: 2, name: 'Jane' },
]
const john = users.find((user) => user.name === 'John')
const johnIndex = users.findIndex((user) => user.name === 'John')
```

## Push, Pop, Shift, and Unshift

These methods modify the original array by adding or removing elements from either end. They're essential for stack and queue operations in JavaScript:

```javascript
const fruits = ['apple']
fruits.push('banana') // Add to end: ['apple', 'banana']
fruits.pop() // Remove from end: ['apple']
fruits.unshift('orange') // Add to start: ['orange', 'apple']
fruits.shift() // Remove from start: ['apple']
```

## Slice and Splice

Two powerful methods for array manipulation with different purposes:

```javascript
// Slice: Extract part of array without modifying it
const numbers = [1, 2, 3, 4, 5]
const subset = numbers.slice(1, 3) // [2, 3]

// Splice: Change array by removing/replacing elements
const letters = ['a', 'b', 'c']
letters.splice(1, 1, 'x') // Replaces 'b' with 'x'
```

## forEach Method

Execute code for each array element without creating a new array. This method is perfect for performing operations that don't require a return value, such as logging or updating external variables:

```javascript
const items = ['book', 'pen', 'desk']
items.forEach((item, index) => {
  console.log(`Item ${index + 1}: ${item}`)
})
```

## Some and Every Methods

Test array elements using boolean conditions:

- `some()`: Checks if at least one element passes the test
- `every()`: Checks if all elements pass the test

```javascript
const scores = [75, 80, 95]
const hasPassingScore = scores.some((score) => score >= 90) // true
const allPassed = scores.every((score) => score >= 70) // true
```

## Practical Use Cases

### Data Transformation

Real-world examples of combining array methods to handle complex data operations:

```javascript
const users = [
  { name: 'John', age: 30 },
  { name: 'Jane', age: 25 },
]

const userNames = users.map((user) => user.name)
const adults = users.filter((user) => user.age >= 18)
```

### Chaining Methods

Array methods can be chained together for complex operations, making your code more readable and maintainable:

```javascript
const numbers = [1, 2, 3, 4, 5, 6]
const sumOfDoubledEvens = numbers
  .filter((num) => num % 2 === 0) // Get even numbers
  .map((num) => num * 2) // Double them
  .reduce((sum, num) => sum + num, 0) // Sum the results
```
